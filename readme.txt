Enable access multiple domain.

== Description ==

= English =
If you have multiple domains and lead more reluctant Wordpress installation, you must activate this plugin. With this plugin, you enable the multiple use of the domain on a Wordpress installation.

Advantages:
The updates for themes, plugins and Wordpress core need not thereby be made more frequently.


= German =
Falls Sie mehrere Domains haben und ungern mehrere Wordpress Installation führen, dann müssen Sie dieses Plugin aktivieren. Mithilfe diesem Plugin aktivieren Sie die mehrfache nutzung der Domain in einer Wordpress Installation.

Vorteile:
Die Updates für Themes, plugins and Wordpress Core brauchen dadurch nicht häufiger gemacht werden.

== Installation ==
1. Create backup of your Wordpress Core Files
2. Upload the zip file to the `/wp-content/plugins/` directory
3. Unzip
4. Activate the plugin in Admininterface
5. Read the instruction about this Plugin in your Admininterface
